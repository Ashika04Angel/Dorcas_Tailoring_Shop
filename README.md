Dorcas Tailoring Shop
A specialized web application for managing tailoring records, billing, and customer history.

Features
Live Search: Instant suggestions for customers by Name or Phone.
Billing System: Generate and manage tailoring bills with specific item details.
Customer Management: Track customer measurements and contact info.
Java Backend: Built with Servlets for robust data handling.

Tech Stack
Frontend: HTML5, Tailwind CSS, JavaScript
Backend: Java Servlets
Version Control: Git & GitHub
